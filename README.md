# 🌐 Hızlı Cari – Promotional Website

A modern, responsive promotional website built with **React** for the mobile app **Hızlı Cari**. Designed to showcase the app’s features, screenshots, user reviews, and download links in a clean and professional layout.

This project focuses purely on the **web experience**, including UI/UX design, layout responsiveness, and modular code structure.

---

## ✨ Website Highlights

- 🎯 Hero section with app branding and CTA
- 🧩 Features overview section with icons and descriptions
- 📸 App screenshots grid (structured with data)
- ⭐ User reviews slider
- 📞 Contact section and developer info in the footer
- 🔗 External links (official site, privacy policy, download page)
- 📱 Fully responsive and optimized for all screen sizes

---

## 🛠️ Built With

- [React](https://reactjs.org/) – UI development
- [CSS Modules](https://github.com/css-modules/css-modules) – Component-level styling
- [MVVM Architecture](https://en.wikipedia.org/wiki/Model–view–viewmodel) – Clean separation of data, views, and logic
- Manual asset optimization and responsive layout tuning

---

| Section     | Description                                 |
| ----------- | ------------------------------------------- |
| Hero        | Brand intro with logo and Google Play badge |
| Features    | Grid of feature cards with icons            |
| Screenshots | Showcase of app UI with responsive grid     |
| Reviews     | Sliding card-style user testimonials        |
| Footer      | Developer info, contact, links              |

## 📦 Installation

```bash
git clone https://github.com/sevderk/GlowCart.git
cd GlowCart
npm install
npm install axios
npm run dev
```

#### ⚠️ Requirements
- Node.js (v16+ recommended): https://nodejs.org
- Git: https://git-scm.com

#### 🌐 View in Browser
- After running npm run dev, open your browser and go to:
- http://localhost:5173
- Or check the terminal output for the exact port (might vary).

#### 📝 Notes
- This project uses Vite for development.
- If npm run dev doesn't work, try:

```bash 
npm install vite --save-dev
```
